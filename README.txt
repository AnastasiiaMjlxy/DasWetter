Програма "DasWetter" є простим додатком для прогнозування опадів на підставі введених даних про температуру,
вологість та швидкість вітру. Користувачу пропонується ввести місцезнаходження, а також дані про температуру,
вологість та швидкість вітру. Далі програма використовує метод лінійної регресії для прогнозування опадів на основі
введених даних. Прогноз розраховується для кількох днів, вказаних користувачем, і виводиться на екран.
Програма також підтримує можливість вибору місцезнаходження для прогнозування опадів.
Наразі програма підтримує два місця: "cityA" і "cityB".
Користувач може ввести одне з цих місцезнаходжень, і програма використовуватиме відповідні дані про
температуру та опади для цього місця при розрахунку прогнозу.
Таким чином, встановлення місцезнаходження заздалегідь дозволяє програмі залежно від введеного місця
отримувати відповідні дані про температуру та опади і використовувати їх для прогнозування опадів для заданого місця.
В майбутньому можна додати більшу кількість міст або використовувати інший метод для реалізації зчитування даних про локацію, але так як
в завданні не було зазначено яким саме способом користуватись, я вирішила використати найпростіший.
